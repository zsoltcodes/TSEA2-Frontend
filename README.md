# Setup Instructions

## Installation

- Pull the remote repository with Git
- Open the folder in VSCode
- EITHER: Run the project with the 'Live Server' extension in VSC.
- OR: Run the project with 'npx serve' if node is installed.

## Project Notes

- Include "<script type="module" src="../js/loadComponents.js"></script>" at the very bottom in the body element in every web page created.
